# miniflare API

> **Local simulator — no Cloudflare management API**
>
> Miniflare is a fully-local simulator for developing and testing Cloudflare Workers offline.
> It does not have Cloudflare platform management API endpoints; it runs entirely on your machine.
>
> For Miniflare's programmatic API (Node.js/Bun) see the official docs:
> https://miniflare.dev/get-started/api
>
> To manage live Cloudflare resources via API, use the Cloudflare Code Mode MCP —
> see `.agents/tools/mcp/cloudflare-code-mode.md`.
