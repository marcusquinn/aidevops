# AI Coding Tools - Supported by aidevops

<!-- AI-CONTEXT-START -->

## Quick Reference

- Primary: OpenCode TUI, OpenCode Desktop, OpenCode extension (Zed/VSCode/AntiGravity/forks)
- Companion: claude-code CLI (called from within OpenCode for sub-tasks)
- Config: `~/.config/opencode/opencode.json`, `~/.claude/`
- Setup: `aidevops update` deploys agents, configures OpenCode MCPs
- Marketplace: aidevops is available in the Claude marketplace
<!-- AI-CONTEXT-END -->

## Supported Tools

aidevops is developed, tested, and supported exclusively with OpenCode and the claude-code CLI.

### OpenCode (Primary)

| Variant | Description |
|---------|-------------|
| **OpenCode TUI** | Terminal UI - keyboard-driven, full MCP support |
| **OpenCode Desktop** | Desktop app with the same capabilities |
| **OpenCode Extension** | Extension for Zed, VSCode, AntiGravity, and other VSCode forks |

- **Installation**: `curl -fsSL https://opencode.ai/install | bash`
- **Configuration**: `~/.config/opencode/opencode.json` or project `.opencode/`
- **Agents**: Generated by `setup.sh` into `~/.config/opencode/agent/`
- **MCPs**: Configured per-agent via YAML frontmatter, managed by setup.sh
- **Server mode**: See `opencode-server.md` for programmatic API access
- **Docs**: See `tools/opencode/opencode.md` for full reference

### claude-code CLI (Companion)

The `claude` CLI from Anthropic is used as a companion tool, called from within OpenCode for sub-tasks and headless dispatch.

- **Usage**: `claude -p "subtask"` (spawns from OpenCode)
- **Config**: `~/.claude/`, `~/.config/Claude/`
- **Docs**: See `claude-code.md`

### OpenClaw (Multi-Channel Assistant)

Personal AI assistant accessible via WhatsApp, Telegram, Slack, Discord, Signal, iMessage, Teams.

- **Installation**: `npm install -g openclaw@latest && openclaw onboard --install-daemon`
- **Docs**: See `openclaw.md`

## Collaborator Compatibility

aidevops generates lightweight pointer files (`.cursorrules`, `.windsurfrules`, etc.) in projects initialized with `aidevops init`. These contain a single reference to `AGENTS.md`, helping collaborators using other editors find the project context. aidevops does not install into, configure, or manage those tools.
