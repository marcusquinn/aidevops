# Creative Production System

> Systems create consistency. Consistency creates results.

## Volume Requirements

| Monthly Spend | New Concepts/Week | Iterations/Week | Total/Week |
|---------------|-------------------|-----------------|------------|
| <$5K | 2-3 | 2-3 | 4-6 |
| $5-20K | 4-6 | 4-6 | 8-12 |
| $20-50K | 6-10 | 6-10 | 12-20 |
| $50-100K | 10-15 | 10-15 | 20-30 |
| $100K+ | 15-25+ | 15-25+ | 30-50+ |

**Pipeline split:** 70% iterations (variations of winners) / 30% new concepts.

**Win rate expectations:**

| Creative Type | Expected Win Rate |
|---------------|-------------------|
| Untested concepts | 10-20% |
| Research-based | 20-30% |
| Iterations of winners | 30-50% |
| AI-optimised variations | 25-40% |

Expect to test 5-10 concepts to find 1-2 winners.

## Production Pipeline

```
1. IDEATION  → Review winners, competitor research, customer feedback, brainstorm angles
2. BRIEF     → Define concept, script/outline, technical specs, assign creator
3. PRODUCTION→ Shoot/create, edit, add captions, export all formats
4. QA        → Check specs, review messaging, compliance check, approve/revise
5. LAUNCH    → Upload to Ads Manager, add to testing campaign, set up tracking, document
```

**Weekly rhythm:**

| Day | Activity |
|-----|----------|
| Mon | Review last week's performance, kill losers |
| Tue | Creative ideation session |
| Wed | Brief creators, start production |
| Thu | Continue production, early QA |
| Fri | Launch new creative |

**Monthly rhythm:** Week 1 & 3 = new concept testing. Week 2 = iterate Week 1 winners. Week 4 = iteration + review/planning.

## Iteration Process

**Winner analysis — ask for each winner:**
1. Hook: what made people stop? what question/statement worked?
2. Emotion: fear, desire, curiosity? how did visuals support it?
3. Proof: testimonial, data, authority? what built trust?
4. Format: video vs static, length, aspect ratio?
5. Audience: age, gender, interest — what does this tell us?

**Variation types:**

| Type | What to Change |
|------|----------------|
| Hook swap | Same body, different hook |
| Visual swap | Same script, different creator/visuals |
| Format swap | Same message, different format (static→video) |
| Length swap | Same concept, shorter/longer |
| Offer swap | Same ad, different offer/CTA |
| Angle shift | Same benefit, different angle |

**Mashup method:** Combine elements from different winners — e.g. A's hook + B's proof section + C's CTA.

**Refresh triggers:**
- CTR declining 20%+ week-over-week
- Frequency >3.0 (prospecting) or >5.0 (retargeting)
- Creative running unchanged 3+ weeks
- CPA rising despite stable CPM

**Refresh process:** Create 2-3 iterations → add to same ad set → let algorithm choose → pause original when new creative wins.

## Team / Resource Options

| Option | Best When | Pros | Cons |
|--------|-----------|------|------|
| In-house | $20K+/month, constant flow, brand requirements | Control, speed, product knowledge, lower per-unit cost at scale | Fixed costs, limited perspectives, capacity constraints |
| Agency | New to paid ads, need strategy, lack capability | Diverse experience, scalable, strategic guidance | Higher cost, less control, slower feedback |
| Freelancers | Specialist tasks, flexible capacity | Cost-effective, diverse perspectives | Management overhead, variable quality, availability |

**UGC creator platforms:** Billo, Insense, Trend, Minisocial. Also: social search, direct DM outreach, creator agencies.

**UGC management checklist:** Clear brief → reference examples → 1-2 revision rounds → usage rights agreement → payment terms.

**Editor workflow:** `Raw footage → Editor → V1 → Review → Revisions → Final → QA → Export`

**Editor checklist:** correct aspect ratios · captions added and timed · sound levels normalised · brand guidelines · file naming convention.

**Creative Strategist** (hire at $50K+/month): analyses performance data, identifies winning patterns, creates briefs, manages pipeline, tests and iterates.

## Technical Specifications

**Video:**

| Spec | Value |
|------|-------|
| Resolution | 1080×1920 (9:16) minimum |
| Format | MP4 or MOV |
| Codec | H.264 |
| Frame rate | 30fps |
| Length | 15-60 seconds |
| Max file size | 4GB |

**Image:**

| Spec | Value |
|------|-------|
| Resolution | 1080×1080 (1:1) minimum |
| Format | JPG or PNG |
| Max file size | 30MB |
| Text overlay | <20% of image area |

**Aspect ratios:**

| Ratio | Dimensions | Best For |
|-------|------------|----------|
| 1:1 | 1080×1080 | Feed (universal) |
| 4:5 | 1080×1350 | Feed (mobile-optimised) |
| 9:16 | 1080×1920 | Stories, Reels |
| 16:9 | 1920×1080 | In-stream, desktop |

## QA Checklist

**Technical:** correct dimensions · clear audio (no clipping) · readable text (mobile test) · captions accurate · no tool watermarks.

**Creative:** hook compelling · message clear in 3 seconds · benefit obvious · CTA clear · brand present.

**Compliance:** no prohibited claims · testimonials are real · no competitor trademark misuse · age/content appropriate · landing page matches ad.

**Phone test (before launch):**
1. Watch without sound — does it work?
2. Watch at 2× speed — is hook strong enough?
3. Show to non-marketer — do they understand?
4. Show to target customer if possible.
