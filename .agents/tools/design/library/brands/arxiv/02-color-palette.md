# arXiv academic: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FFFFFF` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#111111` | Primary text |
| muted | `#4B5563` | Secondary text |
| outline | `#CFCFCF` | Borders and rules |
| primary | `#8B1A1A` | Links, accents, active states |
| primary-container | `#F4E8E8` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
