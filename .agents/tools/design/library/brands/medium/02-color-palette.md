# Medium essay: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FFFFFF` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#242424` | Primary text |
| muted | `#6B6B6B` | Secondary text |
| outline | `#E5E5E5` | Borders and rules |
| primary | `#1A8917` | Links, accents, active states |
| primary-container | `#E8F3E8` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
