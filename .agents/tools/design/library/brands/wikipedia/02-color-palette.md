# Wikipedia reference: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FFFFFF` | Page canvas |
| surface | `#F8F9FA` | Cards and panels |
| on-surface | `#202122` | Primary text |
| muted | `#54595D` | Secondary text |
| outline | `#A2A9B1` | Borders and rules |
| primary | `#0645AD` | Links, accents, active states |
| primary-container | `#EAECF0` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
