# Design System: Clay

Clay — warm, playful B2B craft: artisanal cream canvas, named swatch palette (Matcha/Slushie/Lemon/Ube/Pomegranate/Blueberry), Roobert with 5 OpenType sets, and signature hover animations (rotateZ(-8deg) + hard offset shadow).

## Quick Reference

- **Background**: Warm Cream (`#faf9f7`) — non-negotiable, the warmth is the identity
- **Text**: Clay Black (`#000000`); Secondary: Warm Silver (`#9f9b93`)
- **Border**: Oat Border (`#dad4c8`) — warm, never neutral gray
- **Font**: `Roobert` with OpenType `"ss01 ss03 ss10 ss11 ss12"` on headings; `Space Mono` for code
- **Weights**: 600 headings / 500 UI / 400 body — strict roles
- **Hover**: `rotateZ(-8deg)` + `translateY(-80%)` + hard shadow `rgb(0,0,0) -7px 7px`
- **Radius**: 4px inputs → 12px cards → 24px feature cards → 40px sections → 1584px pills
- **Swatches**: Matcha (`#078a52`), Slushie (`#3bd3fd`), Lemon (`#fbbd41`), Ube (`#43089f`), Pomegranate (`#fc7981`), Blueberry (`#01418d`)
