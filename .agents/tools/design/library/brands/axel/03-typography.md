# Axel editorial evidence: Typography

| Role | Stack |
|---|---|
| Headings | `Newsreader, "Source Serif 4", Georgia, serif` |
| Body | `"IBM Plex Sans", Inter, system-ui, sans-serif` |
| Code/data | `"IBM Plex Mono", "JetBrains Mono", "SFMono-Regular", Consolas, monospace` |

Commercial or platform-specific source fonts are replaced with open-source/system substitutes. Keep body text at 16px+ for HTML and 10.5pt+ for PDF.
