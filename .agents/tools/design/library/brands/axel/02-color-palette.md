# Axel editorial evidence: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FAFAF7` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#0C0F15` | Primary text |
| muted | `#4B5563` | Secondary text |
| outline | `#CDCFC9` | Borders and rules |
| primary | `#2D4BB5` | Links, accents, active states |
| primary-container | `#DBE1F2` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
