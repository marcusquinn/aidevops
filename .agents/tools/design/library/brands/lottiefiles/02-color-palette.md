# LottieFiles motion: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F7FFFD` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#0F172A` | Primary text |
| muted | `#475569` | Secondary text |
| outline | `#CFE8E2` | Borders and rules |
| primary | `#00A58E` | Links, accents, active states |
| primary-container | `#DDFBF5` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
