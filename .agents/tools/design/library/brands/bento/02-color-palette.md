# Bento email: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FFF8F5` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#211A16` | Primary text |
| muted | `#63554D` | Secondary text |
| outline | `#F1D4C4` | Borders and rules |
| primary | `#DB2777` | Links, accents, active states |
| primary-container | `#FCE7F3` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
