# SuperX social analytics: Components

- Report cover and meta grid
- Sticky table of contents with compact PDF affordance
- Evidence badges using report-proportional radii
- Source cards and source-list items with visible link affordances
- Tactic cards, callouts, action prompts, and copyable code blocks
- Fact tables with fixed layout, clipped rounded corners, and print-safe wrapping
- Good/bad examples, priority groups, and owner/recommendation tables

Tables must use wrapping to avoid owner/recommendation overlap in PDF. Links, source rows, and appendix pills must show hover/focus feedback.
