# Polymarket Times: Layout

Use a single HTML preview with a main report column and side table of contents. PDF profile controls page size: A4 default, Letter optional, 16:9 landscape for presentation PDFs only.

## Newspaper layout cues

- Start reports with a masthead-like cover: large centered Playfair Display title, subtitle/edition metadata, and black rules.
- Use sharp section breaks and chapter heroes that feel like newspaper section banners.
- Keep report content width normalized with other styles; do not create a full-width website clone.
- Use dense but readable tables; allow wrapping rather than horizontal overflow in PDF.
- Prefer monochrome imagery/diagrams and market-data accents when examples require visual placeholders.
