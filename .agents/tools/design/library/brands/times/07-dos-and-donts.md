# Polymarket Times: Dos and Don'ts

## Do

- Use Playfair Display for headings, Georgia for body copy, and Menlo for ticker/data/code UI.
- Preserve citations, source IDs, and evidence badges.
- Use `#1A1A1A` or black for long-form text and structural rules.
- Use market green/red only for market/status meaning.
- Test tables and thick rules in PDF.

## Don't

- Create separate A4/Letter HTML files.
- Use green/red decorative accents for body text without labels or contrast checks.
- Round cards or add heavy shadows.
- Treat this educational preset as ownership of the referenced brand.
