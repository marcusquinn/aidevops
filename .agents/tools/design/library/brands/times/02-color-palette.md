# Times newspaper: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F7F2E8` | Page canvas |
| surface | `#FFFDF8` | Cards and panels |
| on-surface | `#111111` | Primary text |
| muted | `#525252` | Secondary text |
| outline | `#C8BFAE` | Borders and rules |
| primary | `#8A2C2C` | Links, accents, active states |
| primary-container | `#F0E1D8` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
