# Polymarket Times: Components

- **Cover/masthead**: centered Playfair Display title, cream paper, top/bottom black rules, edition-style metadata in Menlo/Georgia.
- **Sticky TOC**: square newsprint panel, black border, compact Menlo PDF button, active section marked with market green.
- **Evidence badges**: square/rectangular newspaper labels; value colour still carries Verified/Partial/Inferred/Missing state.
- **Source cards**: bordered clipping-style cards with visible link affordances and dotted separators.
- **Tactic/action cards**: editorial boxes with strong black border and market-green action emphasis.
- **Fact tables**: sharp bordered tables, dotted row separators when possible, numeric/data cells in Menlo.
- **Code blocks**: black ticker-like slabs with Menlo Bold feel, green accent, and visible copy feedback.
- **Good/bad examples**: preserve text labels; use green/red only as supporting state colour.
- **Priority groups**: section-rule headers and compact owner/recommendation tables.

Tables must wrap safely and remain legible in PDF. Prefer borders and rules over shadows.
