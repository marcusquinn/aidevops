# Lifee calendar: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F4FBF8` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#10231C` | Primary text |
| muted | `#496157` | Secondary text |
| outline | `#CFE5DC` | Borders and rules |
| primary | `#047857` | Links, accents, active states |
| primary-container | `#DDF7EC` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
