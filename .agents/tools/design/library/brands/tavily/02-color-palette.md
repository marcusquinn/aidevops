# Tavily AI search: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F6FFFC` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#06251E` | Primary text |
| muted | `#31524A` | Secondary text |
| outline | `#B9D8D0` | Borders and rules |
| primary | `#008A6A` | Links, accents, active states |
| primary-container | `#DDF7EF` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
