# WPCodeBox developer: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F7FAFC` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#111827` | Primary text |
| muted | `#4A5568` | Secondary text |
| outline | `#CBD5E1` | Borders and rules |
| primary | `#047857` | Links, accents, active states |
| primary-container | `#DFF7EC` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
