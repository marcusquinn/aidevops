# Scalefusion enterprise: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F4F7FB` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#111827` | Primary text |
| muted | `#4B5563` | Secondary text |
| outline | `#D6DEE8` | Borders and rules |
| primary | `#174EA6` | Links, accents, active states |
| primary-container | `#E5EFFB` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
