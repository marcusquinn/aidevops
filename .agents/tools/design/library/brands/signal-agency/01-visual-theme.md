# Signal Agency — Visual Theme

## Brand idea

Signal Agency is an evidence-first agency report style for AI-search audits. It should read like research and operational intelligence, not marketing. The page uses warm paper, black ink, typographic hierarchy, and measured rules to create trust.

## Core principles

1. **Signal, not noise** — one warm accent marks attention; everything else is paper, ink, state, or rule.
2. **Research dossier over sales deck** — source IDs, serial codes, evidence states, confidence bars, and metadata are first-class visual elements.
3. **Rules create structure** — thick masthead/footer bars, section hairlines, dotted subdivisions, and table rules replace gradients and decorative cards.
4. **Data remains sober** — tabular numerals, mono labels, and square markers make figures scannable without dashboard gloss.
5. **Components earn colour** — state hues appear only where they encode meaning.

## Signature cues

- Warm grey paper with optional 24px dotted grid.
- Sticky masthead with 4px ink top rule, mono metadata, and a small signal dot in the logo.
- Huge Bricolage Grotesque cover title with an italic terracotta word.
- Section headers with mono section number at left and compact, high-impact headline at right.
- Square, rule-bound components with no conventional card radius.
- Dossier chrome: LEDs, serials, stamps, tabs, source IDs, and provenance footers.

## Tone

Precise, editorial, decisive, and calm. Copy should use concrete evidence words: verified, partial, inferred, missing, source, owner, due, captured, priority, rerun, compare.
