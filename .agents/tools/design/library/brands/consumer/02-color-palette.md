# Consumer advice: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F5F8FB` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#102033` | Primary text |
| muted | `#4C5F73` | Secondary text |
| outline | `#D7E0EA` | Borders and rules |
| primary | `#0057A8` | Links, accents, active states |
| primary-container | `#E4F0FA` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
