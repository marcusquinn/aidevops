# SavvyCal friendly: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FFF8F1` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#2A211B` | Primary text |
| muted | `#66584D` | Secondary text |
| outline | `#E6D7C7` | Borders and rules |
| primary | `#B45309` | Links, accents, active states |
| primary-container | `#FDEAD7` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
