# Ulysses help: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FBFAF7` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#202124` | Primary text |
| muted | `#5F6368` | Secondary text |
| outline | `#DED8CC` | Borders and rules |
| primary | `#A15C2F` | Links, accents, active states |
| primary-container | `#F4E8DF` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
