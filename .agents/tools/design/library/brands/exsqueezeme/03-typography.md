- Space Grotesk-like uppercase display from screenshot review
- Space Mono-like compact support copy from screenshot review

## Report font mapping

- Headings: `Space Grotesk, Inter, ui-sans-serif, system-ui, sans-serif`
- Body: `Space Mono, ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace`

## Report typography requirements

- Use uppercase display energy for headings while keeping report-normalized sizes.
- Use monospace body copy sparingly but consistently for the brand preset.
- Keep code blocks wrapped and readable in PDF.
