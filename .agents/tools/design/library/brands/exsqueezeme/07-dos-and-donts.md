# ExSqueezeMe macOS video: Dos and Don'ts

## Do

- Use black/dark surfaces, white text, and orange CTA slabs.
- Keep square borders and offset white shadows.
- Preserve citations, source IDs, evidence badges, PDF link, and copy-button feedback.
- Test tables and code blocks in PDF.

## Don't

- Reuse the old mellow yellow palette for current ExSqueezeMe output.
- Round cards or soften the poster-like geometry.
- Create separate A4/Letter HTML files.
- Treat this educational preset as ownership of the referenced brand.
