# ExSqueezeMe macOS video: Layout

Use a single HTML preview with a main report column and side table of contents. PDF profile controls page size: A4 default, Letter optional, 16:9 landscape for presentation PDFs only.

## Layout cues

- Keep a dark, centered hero treatment with square borders.
- Use normalized report content width; do not create full-width marketing pages.
- Keep headings wide enough for readable report wrapping.
- Place action-prompt accordions after action sections.
- Preserve one default `report.html`; light/dark variants are only explicit review outputs.
