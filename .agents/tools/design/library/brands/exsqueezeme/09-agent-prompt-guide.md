# ExSqueezeMe macOS video: Agent Prompt Guide

Render with:

```bash
.agents/scripts/report-render-helper.sh render report.md --template exsqueezeme --pdf-profile a4 --output report.html
```

Prompt style:

> Render this Markdown-first report in the current ExSqueezeMe style: dark black panels, white text and borders, orange slab accents, square geometry, Space Grotesk headings, Space Mono body/code UI, compact PDF button, copy-button tick feedback, and PDF-safe tables.

For slides, keep `report.html` as the preview and export PDF with `--pdf-profile slides-16-9-1`, `slides-16-9-2`, or `slides-16-9-3`.
