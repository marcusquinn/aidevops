# Zeroheight design docs: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FAFAFC` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#171717` | Primary text |
| muted | `#52525B` | Secondary text |
| outline | `#E4E4E7` | Borders and rules |
| primary | `#7C3AED` | Links, accents, active states |
| primary-container | `#F0E7FF` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
