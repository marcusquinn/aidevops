# Indexsy agency: Dos and Don'ts

## Do

- Preserve citations, source IDs, and evidence badges.
- Use `#F9FAFB`/`#D7DBE3` for long-form text on the dark background.
- Use blue/violet for CTA, links, active TOC, and PDF/action affordances.
- Use yellow only for small numbered markers or counter accents.
- Test tables in PDF.

## Don't

- Create separate A4/Letter HTML files.
- Use accent colours for body text without contrast checks.
- Revert to the old orange/light palette for Indexsy reports.
- Treat this educational preset as ownership of the referenced brand.
