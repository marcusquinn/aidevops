# Indexsy agency: Components

- Report cover and meta grid
- Sticky table of contents
- Top-right TOC PDF pill/button when a sibling PDF export is available
- Evidence badges
- Source cards
- Tactic cards
- Fact tables with fixed layout
- Good/bad examples
- Priority groups and owner/recommendation tables
- Action Prompt accordions beneath action lines/panels with copyable prompt code blocks
- Code, Mermaid, and LaTeX panels using the same dark surface treatment

Tables must use wrapping to avoid owner/recommendation overlap in PDF.

Buttons and badges must keep vertical margin when wrapping to a second line. Avoid placing copy controls over code content; keep copy buttons in the code-block header.
