# Indexsy agency: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FFF7ED` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#111827` | Primary text |
| muted | `#4B5563` | Secondary text |
| outline | `#FED7AA` | Borders and rules |
| primary | `#EA580C` | Links, accents, active states |
| primary-container | `#FFEDD5` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
