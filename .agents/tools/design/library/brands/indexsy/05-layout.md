# Indexsy agency: Layout

Use a single HTML preview with a main report column and side table of contents for rich templates. The `basic` template may remain one-column/plain for debugging. PDF profile controls page size: A4 default, Letter optional, 16:9 landscape for presentation PDFs only.

Indexsy-style reports should feel more like a dark landing-page section than a beige document: broad spacing, rounded cards, large centered cover sections, and clear left/right rhythm where the content allows it.
