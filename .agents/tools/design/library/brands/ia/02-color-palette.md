# iA writing: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F7F7F3` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#111111` | Primary text |
| muted | `#555555` | Secondary text |
| outline | `#D9D9D2` | Borders and rules |
| primary | `#005EA8` | Links, accents, active states |
| primary-container | `#E4F0F8` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
