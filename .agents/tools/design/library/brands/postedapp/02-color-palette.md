# Posted App editorial: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#FFFDF8` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#111111` | Primary text |
| muted | `#555555` | Secondary text |
| outline | `#DDD6C8` | Borders and rules |
| primary | `#0F766E` | Links, accents, active states |
| primary-container | `#E0F2F1` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
