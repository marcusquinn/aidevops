# Ghost documentation: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#F8FAFC` | Page canvas |
| surface | `#FFFFFF` | Cards and panels |
| on-surface | `#15171A` | Primary text |
| muted | `#4B5563` | Secondary text |
| outline | `#DDE3EA` | Borders and rules |
| primary | `#30CF43` | Links, accents, active states |
| primary-container | `#E8FBEA` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
