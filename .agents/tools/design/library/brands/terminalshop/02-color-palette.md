# Terminal Shop CLI: Colour Palette

| Token | Value | Use |
|---|---|---|
| background | `#0B0F0C` | Page canvas |
| surface | `#101810` | Cards and panels |
| on-surface | `#E6F6E6` | Primary text |
| muted | `#A7C7A7` | Secondary text |
| outline | `#284A28` | Borders and rules |
| primary | `#4ADE80` | Links, accents, active states |
| primary-container | `#16321F` | Badge and callout backgrounds |

Use primary text for reading. Accent colours are decorative unless contrast is verified.
