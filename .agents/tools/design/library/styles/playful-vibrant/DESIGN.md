# Design System: Playful Vibrant

Energetic, joyful, expressive. Indigo `#6366f1` primary, rose `#f43f5e` accent, rounded shapes (16–20px), bouncy spring animations. Targets children's education, gaming, social communities, creative tools.

## Quick Reference

**Colours:** Primary `#6366f1` · Primary gradient `135deg #6366f1→#8b5cf6` · Accent `#f43f5e` · Heading text `#1e1b4b` · Body text `#374151` · Background `#FAFAFA`

**Shapes:** Buttons/inputs `border-radius: 16px` · Cards `border-radius: 20px` · Pills/avatars `border-radius: 9999px`

**Animation:** Spring easing `cubic-bezier(0.34, 1.56, 0.64, 1)` · Transition `200ms` · Hover: `translateY(-2px)` + shadow increase

**Typography:** Nunito (rounded sans-serif) · Headings weight 700 · Display weight 800
