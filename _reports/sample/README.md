# Full Sample Reports

`_reports/sample/` contains end-to-end sample reports that are larger than the
compact examples in `_reports/examples/`. Use these to review chaptering,
appendices, source-ledger links, and long-report behaviour before publishing a
new report workflow.

For reports using a DESIGN.md with dark/inverse tokens, include both
`<style>-light.html` and `<style>-dark.html` previews so panel colour inversion
can be reviewed independently of the viewer's OS colour scheme.
