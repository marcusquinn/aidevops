# _campaigns/lib/ — Versioned Campaign Library

<!-- SPDX-License-Identifier: MIT -->
<!-- SPDX-FileCopyrightText: 2025-2026 Marcus Quinn -->

Use this directory for reviewed, reusable, public-safe campaign inputs:

- `brand/` — brand identity, palette, voice, and tone references.
- `swipe/` — inspiration files with source and rationale metadata.
- `assets/` — asset manifests and small preview thumbnails.

Do not place competitive intel or in-flight campaign drafts here; those belong
in ignored `intel/` or `active/` directories.
